# MiniProyecto 2: TIC is Among Us y Pokédex

Este repositorio contiene todos los entregables para el MiniProyecto 2

## Contenido del Repositorio

* [Mini_Proyecto_2.pdf](/Mini_Proyecto_2.pdf): El informe completo del proyecto en formato PDF.
* [simon.py](/simon.py): Minijuego físico de memoria "Simón Dice" (botones/audio).
* [supersonico.py](/supersonico.py): Minijuego físico de reflejos (sensor ultrasónico HC-SR04).
* [flappy_pkmn.py](/flappy_pkmn.py): Minijuego virtual de Pokémon con estilo "Flappy Bird".
* [lluvia_pkmn.py](/lluvia_pkmn.py): Minijuego virtual Lluvia de Pokémon
* [main_client.py](/main_client.py): Script maestro que gestiona la conexión SSH/SFTP con el Host Central (servidor del profesor) y lanza los minijuegos.
* [pokedex.json](/pokedex.json): Base de datos en formato JSON utilizada por la Pokédex.
* [pokedex.py](/pokedex.py): Aplicación de Pokédex que controla los LEDs RGB según el tipo de Pokémon.


* Carpeta `Pokemon todas gen/`: Sprites para la aplicación Pokédex.
* Carpeta `sprites/`: Gráficos para el minijuego Flappy Pokémon.
* Carpeta `img/`: Recursos gráficos para el minijuego Lluvia de Pokémon.
